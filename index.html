Правила чата

❗Правила для администратора❗

Главное правило. Админ должен быть ленив. Админ лениво озирается по сторонам, лениво оценивает действия участников, лениво принимает решения. Торопиться некуда, впереди вся жизнь. Поспешность ведет к ошибкам. Ошибки заставляют грустить участников.

❗Правила для участников❗

Главное правило. Участник не должен лениться. Участник не ленится оценивать мнение соседей, участник не ленится понимать просьбы админа, участник не ленится читать правила чата. Участник не ленится делать чат вокруг комфортным не только для себя, но и для остальных участников. Лень ведет к ухудшению чата вокруг участника. Лень заставляет админов применять первое и главное правило.

Дружественные ресурсы

  • Общий чат ЖК "Люблинский парк".
    Здесь общаются жители всего ЖК, есть свои отдельные подчаты про почти все, что можно обсудить про наш ЖК. Проект социальный, в нем нет рекламы, есть правила и модерация. В чате есть несколько ботов, упрощающих жизнь пользователей, главный из них умеет по ключевым словам в сообщении подсказать вам нужную ссылку или контакт в ЖК.
  • Новостной канал ЖК "Люблинский парк".
    Без преуменьшения, лучший новостной канал про наш ЖК. Единственный, который делает авторский контент, публикуя новости с полей. Здесь вы найдете новости об открывающихся в ЖК магазинах, увидите эксклюзивные фото строительства ЖК, узнаете другие полезные вещи. Команда канала проводит викторины, сама при этом являясь реальными соседями по ЖК. На канале есть реклама, помогающая ему вести свою деятельность, но все она связана с непосредственно нашим районом и публикуется строго в соответствии с законодательством РФ.
